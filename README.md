# My HTML Only Website

## Overview

This project is a multi-page website built entirely using **pure HTML**, without any CSS or JavaScript. It was created as part of a web development assignment to demonstrate mastery of core HTML elements and semantic structure. The design was first prototyped using **Figma**, then implemented using only HTML.
